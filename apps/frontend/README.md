## Valora CRM - Frontend

*Interface web do sistema Valora CRM.*

Tecnologias
React, Vite, TailwindCSS, Recharts, Framer Motion, Axios

Funcionalidades
- Login e cadastro
- Dashboard com gráficos
- Pipeline de clientes
- Gerenciamento de clientes

Como Executar
npm install
npm run dev

URL:
http://localhost:5173

Observações
- Backend deve estar rodando na porta 3000
