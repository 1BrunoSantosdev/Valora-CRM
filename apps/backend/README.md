## Valora CRM - Backend

*API responsável pela lógica do sistema.*

## Tecnologias
Node.js, Express, Prisma, PostgreSQL, JWT

Funcionalidades
- Autenticação
- CRUD de clientes
- Multi-empresa

Como Executar
npm install
npm run dev

Servidor:
http://localhost:3000

Observações
- Utiliza JWT
- Dados isolados por companyId
