## Valora CRM

Sistema CRM completo com foco em gestão de clientes, pipeline de vendas e estrutura preparada para evolução em SaaS.

## Visão Geral
O Valora CRM é uma aplicação fullstack que permite gerenciar clientes, acompanhar estágios de negociação e visualizar métricas em tempo real. O projeto foi desenvolvido com foco em simular um produto real de mercado.

## Funcionalidades
- Autenticação de usuários (login e cadastro)
- Isolamento de dados por usuário (multi-empresa)
- CRUD completo de clientes
- Pipeline de vendas (Lead, Negociando, Fechado, Perdido)
- Dashboard com métricas e gráficos dinâmicos
- Atualização periódica dos dados

## Tecnologias Utilizadas
Frontend: React, Vite, TailwindCSS, Recharts, Framer Motion
Backend: Node.js, Express, Prisma ORM, PostgreSQL, JWT

## Como Executar

Backend:
cd backend
npm install
npx prisma db push
npm run dev

Frontend:
cd frontend
npm install
npm run dev
